## 🚀 Como Usar

### Para Clientes (Comprar)
1. Acesse [@xRocketFlow_Bot](https://t.me/xRocketFlow_Bot)
2. Navegue pelo catálogo
3. Adicione ao carrinho
4. Finalize a compra

### Para Lojistas (Vender)
1. Digite `/criar_loja` no bot
2. Adicione seus produtos
3. Personalize sua loja
4. Compartilhe o link `t.me/xRocketFlow_Bot?start=sua_loja`

## 🛠️ Deploy Próprio

### Pré-requisitos
- Node.js 18+
- Conta no Render (ou qualquer host)

### Passos
```bash
# Clone o repositório
git clone https://github.com/orlandojamie833-ux/xRocket-Flow-2030.git

# Instale as dependências
npm install

# Configure as variáveis (crie um arquivo .env)
cp .env.example .env

# Execute
npm start
